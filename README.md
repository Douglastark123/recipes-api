## Instalação

```bash
$ npm install
```

## Executar o app

```bash

$ npm run start

# watch mode
$ npm run dev

# production mode
$ npm run prod
```

## Testar

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
